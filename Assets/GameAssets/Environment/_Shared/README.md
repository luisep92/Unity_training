# Environment Shared Code

Common code used by multiple environment systems (Water, Vegetation, Creatures).

## What goes here:
- ✅ Shared environmental interfaces
- ✅ Common environmental behaviors
- ✅ Environmental reaction event system
- ✅ Utilities specific to environment systems

## What does NOT go here:
- ❌ Truly universal code (use `_Core/`)
- ❌ System-specific code (use the specific module folder)
